# Auto-World-Map

[English](./README.md)

当切换服务器时自动为 VoxelMap, XaeroMap 切换地图。

## 依赖项

| 依赖          | 类型  | 下载                                                     |
|-------------|-----|--------------------------------------------------------|
| Protocolize | 必须  | [Github](https://github.com/Exceptionflug/protocolize) |

## 开发

### 支持

当前主要为最新版 `BungeeCord`, `Waterfall` 开发。

### 文档

英文文档与中文文档是逐行对应的。
